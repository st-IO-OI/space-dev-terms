# ペイロードとは？

- **よみがな：** ぺいろーど  
- **英語名：** Payload  

---

## 概要

ペイロードとは、ロケットや人工衛星などの宇宙機において主目的となる機器や物資の総称である。  

---

## 詳細

ペイロードとは、ロケットや人工衛星などの宇宙機において、主たる運搬目的となる搭載機器や物資の総称のことです。  
具体的には、人工衛星そのもの、観測機器、宇宙探査機、物資、科学実験装置などが含まれます。  
一言でわかりやすく言えば、「宇宙に持っていく荷物」といったところでしょうか。  

ロケットのペイロードというと、軌道上または目的地点に運ばれる主役的な存在です。  
この「ペイロードを宇宙に運ぶこと」がロケットが打ち上げられる目的であると言えます。  

打ち上げ計画を考える際にも、どのようなペイロードを運搬するかについて考えることが重要になります。  
ペイロードの質量や寸法、耐荷重性、冷却要件などがロケット設計や打ち上げ方法に大きく影響します。  

また、宇宙機の開発では、ペイロードと衛星バスをつなぐ部分の設計：インタフェース設計を適切に行うことが極めて重要になります。  
近年では、ペイロードの多様化に対応するために、様々なミッションを同時に行う人工衛星を打ち上げたり、小型衛星を共同で同時に打ち上げたりすることも一般化しています。
