# 固体燃料ロケットとは？

- **よみがな：** こたいねんりょうろけっと  
- **英語名：** Solid Propellant Rocket  

---

## 概要

固体燃料ロケットとは、固体状の燃料を用いて推進力を得るロケットである。

---

## 詳細

固体燃料ロケットは、燃料と酸化剤をあらかじめ混合・成形した固体推進剤を用いて推進力を得る方式のロケットです。  
点火すると一気に燃焼が進み、強力な推力を発生させます。  
構造がシンプルで、可動部が少ないため信頼性が高く、保管や取り扱いも容易です。  

このタイプのロケットは、軍事用途や衛星打ち上げの補助ブースターとして広く使用されています。  
代表的な例として、スペースシャトルの固体補助ロケット（SRB）や日本のイプシロンロケットなどがあります。  

技術的には、燃焼制御が難しく一度点火すると停止や再点火ができないという制約がありますが、即時発射性や低コスト性が評価されています。  
ちなみに、固体燃料ロケットは打ち上げ準備が短時間で済むため、緊急発射や大量配備に適しており、宇宙開発だけでなく防衛分野でも活用されています。
