# 再使用型ロケット

- **よみがな：** さいしようがたろけっと  
- **英語名：** Reusable Rocket  

---

## 概要

再使用型ロケットとは、複数回の打ち上げに対応可能な設計を持つロケットである。

---

## 詳細

再使用型ロケットは、打ち上げ後に機体の一部または全部を回収し再度整備し直すことによって、再び打ち上げに利用することができるロケットのことです。  
従来の使い捨て型ロケットと異なり打ち上げごとに新造する必要がないため、打ち上げコストの大幅な削減が可能となります。  

何度も同じ機体を用いて打ち上げを行うためには、着陸制御に関する技術、多数の打ち上げに対応するための耐熱構造、推進系の再点火機能などといった技術が求められます。  

再使用型ロケットの代表例としては、スペースXのファルコン9があげられます。  
ファルコンロケットは、打ち上げ後に第1段ロケットを垂直着陸させることによってロケットを回収します。  
これを整備した後に再度打ち上げることで、再使用のサイクルを実現しています。  
このロケットの登場は、商業宇宙輸送の世界に革命をもたらした出来事であったと言えるでしょう。  

再使用型ロケットは環境負荷の低減や打ち上げ頻度の向上にも貢献するため、今後の宇宙開発において重要な役割を果たすと期待されています。
