# 気象衛星とは？

- **よみがな：** きしょうえいせい  
- **英語名：** Weather Satellite  

---

## 概要

気象衛星とは、地球の気象や気候の観測を目的として運用される人工衛星である。

---

## 詳細

気象衛星は、地球の大気、雲、降水、風、海面温度などの情報を収集するために設計された衛星です。  
これらの衛星は、地球の周回軌道上から広範囲の気象データを取得し、天気予報や気候研究、防災活動に活用されています。  

主に静止軌道衛星と極軌道衛星の2種類があり、静止軌道衛星は赤道上空約36,000kmに位置し、同じ地域を常に観測できます。  
一方、極軌道衛星は地球の南北を周回しながら、地球全体を網羅的に観測することが可能です。  

搭載される観測機器には、可視光・赤外線センサー、マイクロ波放射計などがあり、これらを用いて雲の動きや気温、水蒸気量などを測定します。  
取得されたデータは、地上局に送信され、解析を経て気象庁や研究機関に提供されます。  

気象衛星は、台風や豪雨などの災害予測において極めて重要な役割を果たしており、早期警戒や避難指示の根拠となる情報を提供します。  
また、地球温暖化や気候変動の長期的な監視にも活用されており、国際的な協力のもとで運用されています。  

日本では「ひまわり」シリーズが代表的な気象衛星です。  
最新の「ひまわり8号」「ひまわり9号」は高頻度・高解像度の観測を可能にしており、現代の気象予報技術を支える存在となっています。
