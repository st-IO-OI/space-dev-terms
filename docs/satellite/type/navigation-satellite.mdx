# 測位衛星

- **よみがな：** そくいえいせい  
- **英語名：** Navigation Satellite  

---

## 概要

測位衛星とは、地球上の位置情報を提供するために中軌道などに配置された衛星であり、位置情報や時刻同期に利用される。

---

## 詳細

測位衛星は、地球上の任意の地点における位置、速度、時刻情報を提供するための人工衛星です。  
代表的なシステムには、米国のGPS、ロシアのGLONASS、欧州のGalileo、日本の準天頂衛星システム（QZSS）などがあります。  

これらの衛星は中軌道に配置され、地球全体をカバーするように複数機が運用されています。  
地上の受信機は、複数の衛星からの信号を受信し、三角測量の原理により位置を算出します。  

測位衛星は、カーナビゲーション、スマートフォン、航空機、船舶、農業機械など、幅広い分野で活用されています。  
また、災害時の位置特定や、精密な時刻同期にも利用され、社会インフラの一部として機能しています。  

技術的には、高精度な原子時計、信号の安定性、軌道維持のための制御技術が重要です。  
今後は、より高精度な測位や、都市部での利用拡大に向けた技術開発が進められています。
