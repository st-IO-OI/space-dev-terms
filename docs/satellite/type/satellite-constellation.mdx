# 衛星コンステレーション

- **よみがな：** えいせいこんすてれーしょん  
- **英語名：** Satellite Constellation  

---

## 概要

衛星コンステレーションとは、複数の人工衛星を連携させて運用し、広範囲かつ高頻度な観測や通信を実現するシステムである。

---

## 詳細

衛星コンステレーションは、地球周回軌道上に多数の人工衛星を配置し、それらをネットワークとして連携させることで、地球全体をカバーする観測・通信システムを構築する技術です。  
この方式により、従来の単独衛星では難しかった高頻度のデータ取得や全球の網羅などが可能となります。  

代表的な例として、SpaceXの「Starlink」などの通信衛星群、Planet Labsの地球観測衛星群などが挙げられます。  
これらは、インターネット接続の地球規模での提供や、農業・環境モニタリング、災害対応などに活用されています。  

技術的には、衛星間通信（ISL）や自律運用、軌道管理などの高度な制御が求められます。  
また、打ち上げコストの低下や小型衛星の普及が、コンステレーションの実現を後押ししています。  

一方で、軌道上の混雑や衝突リスク、スペースデブリの増加といった課題もあり、国際的なルール整備が進められています。  
衛星コンステレーションは、次世代の宇宙インフラとして注目される分野です。
