# 宇宙開発に必要なコト

みなさんは「宇宙開発」についてどのようなイメージを持っているでしょうか？

ロマンがある、夢の技術、憧れの世界、、、  
などといった、美しいイメージを持っている人がたくさんいると思います。  

その一方で、
難しそう、ハードルが高そう、頭が良くないと無理、、、
など、自分がその役割を担うイメージが持てない人が多いかもしれません。  

しかし、近年宇宙業界は大きく変化しています。  
これからはより宇宙が身近な世界になっていくことでしょう。  

その中でより多くの人の協力が必要になっていくことになります。 
 
宇宙開発は、一部の専門家だけが行う特別な世界ではありません。  
理系・文系、技術者・研究者だけでなく、企画、デザイン、広報、法務、経営など、あらゆる分野の知識と協力が求められています。  

この広大なプロジェクトは、ひとりの力で成し遂げられるものではなく、様々な能力や経験を持つ多くの人々が集まって初めて実現します。  
だからこそ、宇宙に関わりたいと思うあなたにも、必ず活かせる場所があります。  

宇宙開発の技術や専門知識は確かに高度ですが、その一方で多様な視点や新しいアイデアも必要です。  
難しそうに見えるかもしれませんが、まずは興味を持ち、一歩を踏み出すことが大切です。  

このページを通じて、「宇宙開発は遠い世界」ではなく、「誰もが参加できる未来の創造」であることを伝えたいと思います。  
あなたの力が、次の宇宙の一歩を支えるかもしれません。
