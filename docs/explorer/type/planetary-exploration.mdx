# 惑星探査

- **よみがな：** わくせいたんさ  
- **英語名：** Planetary Exploration  

---

## 概要

惑星探査とは、地球以外の惑星や衛星に対して行われる科学的・技術的な調査活動である。

---

## 詳細

惑星探査は、太陽系内外の惑星やその衛星に対して、探査機を用いて行われる調査活動です。  
その目的は、惑星の起源や進化、地質構造、大気の性質、生命の可能性などを明らかにすることにあります。  
実際に、人類は長年にわたり火星、金星、木星などの惑星への探査を通じて多くの知見を得てきました。  

惑星探査には、フライバイ（接近通過）、ランデヴー（滞在）、着陸など、さまざまな方式があります。  
フライバイは探査機が目標天体の近くを高速で通過しながら観測を行う方法で、目標天体に到達しやすいという利点があります。  
また、ランデヴーは探査機が天体の周囲に長期間滞在する方法で、継続的に観測を行える点が強みです。  
そして、着陸とは探査機が天体の表面に降り立つ探査手法です。  
直接的な観測やサンプル採取を行うことができるため、多くの成果を得ることが可能になります。  
これらの方式は、探査の目的や技術レベルに応じて使い分けられています。

実際に惑星探査を実現するためには、長距離通信、精密な軌道制御、耐放射線設計、自律運用などの技術が必要になります。  
惑星探査は、宇宙科学の最前線であり、今後も国際協力のもとで多くのミッションが計画されています。
