# 宇宙生命探査

- **よみがな：** うちゅうせいめいたんさ  
- **英語名：** Astrobiology Exploration  

---

## 概要

宇宙生命探査とは、宇宙空間における生命の存在や起源、進化の可能性を科学的に調査する探査活動である。

---

## 詳細

宇宙生命探査は、地球外生命の存在を探るために、惑星や衛星、彗星、小惑星などを対象として行われる科学的な探査活動です。  
この分野は「アストロバイオロジー（宇宙生物学）」とも呼ばれ、生命の普遍性や宇宙における生存環境の条件を明らかにすることを目的としています。  

代表的な探査対象には、火星、エウロパ（木星の衛星）、エンケラドゥス（土星の衛星）などがあります。  
これらの天体には液体の水や有機物の存在が示唆されており、生命が生存可能な環境が存在するのではないかと言われています。  
今後こうした天体を目指して更なる探査が進められていくことでしょう。  

技術的には、着陸機やローバーによる地表分析、スペクトル観測、氷層の掘削、サンプルリターンなどが用いられます。  
また、生命の痕跡（バイオシグネチャー）を検出するための高感度センサーや分析装置の開発も進められています。  

宇宙生命探査は、生命の起源や宇宙における存在可能性を探る壮大な科学的挑戦であり、人類の宇宙観を根本から変える可能性を秘めています。
