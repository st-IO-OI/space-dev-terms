# 小惑星探査

- **よみがな：** しょうわくせいたんさ  
- **英語名：** Asteroid Exploration  

---

## 概要

小惑星探査とは、太陽系内の小惑星を対象に、探査機を用いてその構造や成分を調査する科学的活動である。

---

## 詳細

小惑星探査は、太陽系に存在する小惑星に対して無人探査機を送り込み、地形や岩石の組成、そしてその起源などを調査する宇宙科学の一分野です。  
小惑星は、惑星形成の過程で残された原始的な天体であり、太陽系の歴史を解明する手がかりを多く含んでいるとされており、  
かねてより探査対象として大きな注目を集めてきました。  

探査手法には、フライバイ、周回、着陸、サンプルリターンなどがあり、目的に応じて選択されます。  
日本の「はやぶさ」シリーズは、世界初の小惑星からのサンプルリターンを成功させたことで国際的に高く評価されています。  
また、NASAの「OSIRIS-REx」も、小惑星ベンヌからのサンプル回収に成功しました。  

小惑星探査は、科学的意義だけでなく、将来的な資源採掘（アステロイドマイニング）や地球衝突リスクの評価にもつながる応用的価値があります。  
そして、探査技術の進展によって、より遠方の小惑星や複数天体への同時探査も可能になりつつあります。  

小惑星探査は、宇宙の起源を探る旅であり、未来の宇宙利用にも直結する重要なミッションです。
